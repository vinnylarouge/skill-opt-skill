skill-opt · explainer

The same optimizer, in space or in time.

Microsoft's SkillOpt improves a skill with four roles — a frozen Target, an Optimizer, a Gate, and a Memory — passing artifacts between them. That's an orchestration: a team of agents working at once, in space.

But anything a team does by passing messages, a single agent can do by swapping hats in sequence — putting one on, doing that role's bit, taking it off, checking the run-directory mailbox, putting on the next. That's a skill: one agent, four hats, in time. The only real difference is parallelism. Flip the toggle and watch the team collapse into one actor.
